◆ Init()

uint8_t IntroSatLib::BaseDevice::Init ( )
virtual

Инициализация устройства на шине I2C.

Возвращает
0, если инициализация завершена успешно
код ошибки HAL_StatusTypeDef, если при инициализации возникла ошибка

Переопределяется в IntroSatLib::Accelerometer, IntroSatLib::AccelerometerV2, IntroSatLib::AK8963, IntroSatLib::BaseFlyWheel, IntroSatLib::Gyroscope, IntroSatLib::GyroscopeV2, IntroSatLib::IRCamera, IntroSatLib::LightSensor и IntroSatLib::MagnetometerV2.
