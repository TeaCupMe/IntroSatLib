◆ integrationX()

float IntroSatLib::GyroscopeV2::integrationX ( )

Получение углового перемещения вокруг оси X с момента предыдущего вызова integrationX()

Заметки
Метод учитывает только два измерения угловой скорости: одно, сделанное в момент вызова этого метода, и второе - предыдущее. Если время между измерениями велико и угловая скорость успела нелинейно измениться - результат будет некорректным.
Возвращает
Угловое перемещение в градусах.
